Actions

Escape From Castle Matsumoto is played using four Directions and a single Action button.

Phase Left/Right Up/Down Action
Roaming Move Player Move Player Default Action
if no action shown, enter Inventory
Inventory Change Items Exit Inventory Select Item
Map - Change Floors Close Map
Lock Abort Locking/Unlocking Set Next Combination Pin Lock/Unlock (once all pins set)
Battle Punch/Block (when Stanced) Punch/Block (when Stanced) Hold to Stance

Battle

  • Hold Action to assume Stance
  • While holding Action, press direction:
    • Towards other player - low kick
    • Away from other player - high block
    • Up - high kick
    • Down - low block
